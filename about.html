Book Manager is a lightweight web application built with Flask, SQLite, and Bootstrap. It allows users to log in, and add, update, and delete books.

The app supports user-based login and role-based book visibility (admin/user), ensuring data privacy and role control.

Whether you're a student, reader, or admin, this tool makes managing books fast and easy.

Developed by Shauryaveer Singh Dabas using modern web technologies.
